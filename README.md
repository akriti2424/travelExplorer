# travelExplorer
Design a "Travel Explorer" landing page that uses Unsplash API to show destination images.

A responsive Travel Destination Landing Page built with React  + Tailwind CSS, using the Unsplash API to display beautiful destination images.


 Features
 Hero section with background image and search bar
 Default destination images on initial load
 Search destinations using Unsplash API
 Responsive image gallery with hover effects
 Image details modal (photographer name, likes)
 Skeleton loader while images are loading
 Smooth UI interactions
 Clean and modern UI with Tailwind CSS


 React (Vite)
 Tailwind CSS
 Unsplash API
 Axios
 Framer Motion (animations)

 Getting Started
1. Clone the repository  
2. Install dependencies  

Create a .env file and add your Unsplash key
VITE_UNSPLASH_KEY= access_key

Start the development server
command: npm run dev
